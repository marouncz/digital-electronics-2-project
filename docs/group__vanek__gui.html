Detailed Description

Graphical user interface library for oled display.

#include <gui.h>
gui.h

The library contains functions for contoling gui for xylophone.

Note
Depended on twi.h - I2C/TWI library for AVR-GCC by Tomas Fryza and oled.h - Michael Köhler's OLED librar.
